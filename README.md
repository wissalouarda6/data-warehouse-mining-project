# Data Warehouse and Data Mining Project

A comprehensive retail sales analysis system implementing ETL processes, OLAP analysis, and data mining techniques in R.

## Features

-  Complete Data Warehouse (Star Schema)
-  OLAP Multidimensional Analysis
-  RFM Customer Segmentation
-  K-means Clustering
-  Anomaly Detection
-  Interactive Visualizations
-  KPI Dashboard
-  Automated CSV Exports

## Requirements

- R version 4.0+
- Base R packages (stats, graphics, utils)

## Installation

```bash
git clone https://github.com/yourusername/data-warehouse-mining-project.git
cd data-warehouse-mining-project
```

## Usage

```r
source("main.R")
```

## Results

The script generates:

- 4 analytical charts
- 4 CSV export files in `exports/` folder
- Complete KPI dashboard in console

## Project Structure

- **Part 1**: Data Warehouse Creation
- **Part 2**: OLAP Analysis
- **Part 3**: Visualizations
- **Part 4**: Data Mining (Clustering & Anomaly Detection)
- **Part 5**: KPI Dashboard
- **Part 6**: Export Results
